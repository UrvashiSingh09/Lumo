## Run Instructions

### Backend
```
cd backend
npm install
node server.js
```

### Frontend
```
cd frontend
npm install
npm start
```

### AI Engine (optional)
```
cd ai_engine
pip install textblob
python -m textblob.download_corpora
python emotion_analysis.py
```
